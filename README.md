# DevOps + AIOps E-Commerce Monitoring Platform

## Features

- Full Stack E-Commerce CRUD
- AI-Powered Log Analysis
- Dockerized Microservices
- Prometheus Monitoring
- Grafana Dashboards
- PostgreSQL Database
- REST APIs
- CI/CD with GitHub Actions
- Real-Time Metrics Tracking

## Tech Stack

Frontend:
- React.js

Backend:
- Flask
- Python

Database:
- PostgreSQL

DevOps:
- Docker
- Docker Compose
- GitHub Actions

Monitoring:
- Prometheus
- Grafana

AI / AIOps:
- Ollama
- Llama 3
