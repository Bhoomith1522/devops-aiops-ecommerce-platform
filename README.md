# Project Overview

This is a full-stack DevOps + AIOps E-Commerce platform built using:

- React Frontend
- Flask Backend
- PostgreSQL Database
- Docker & Docker Compose
- GitHub Actions CI/CD

The application supports product management with full CRUD operations and containerized deployment architecture.

---

# Features

## Backend
- Flask REST API
- PostgreSQL Integration
- CRUD Operations
- Dockerized Backend
- CORS Enabled

## Frontend
- React UI
- Product Listing
- Add Product

- API Integration

## DevOps
- Docker Compose
