# DevOps + AIOps E-Commerce Platform

A full-stack containerized e-commerce platform built using:

- React
- Flask
- PostgreSQL
- Docker
- Docker Compose

## Features

- Product CRUD operations
- REST API backend
- PostgreSQL persistence
- Frontend-backend integration
- Dockerized architecture
- Multi-container orchestration

## Tech Stack

- Frontend: React
- Backend: Flask
- Database: PostgreSQL
- Containerization: Docker
- Orchestration: Docker Compose
